Comandos para se utilizar no GitBashHere

- git init - inicia um repositório
- git status - analiza o status do repositório, se arquivos ainda não foram enviados ou se já foram enviados . 
-git add nome do arquivo que foi alterado - adiciona arquivo na área de staggin.
-git add . - adiciona todos os arquivos criados e modificados na área de stadding.
- git commit -m "primeiro  commit" - dar commit no repositório.
-git remote add origin link do repositório Github.
-git push -u origin master.

Comandos  de Branch

-git reflog - mostra as versões com um número  de id, a mais  atual fica em primeiro.

Navegar entre as versões:
- git reset --hard (nº id da versão que desejo navegar)

- git branch - mostra quais branchs estão ativas

Criar branch:
- git branch (nome da branch) ex: staging
nome comum de branchs que recebem atuaçlizações que ainda não estão testadas, em produção, staging.

Mudar de brach:
*master
staging

A branch com asterisco é a branch atual que está sendo modificada.

Para alterar de branch usar o comando: git checkout staging
